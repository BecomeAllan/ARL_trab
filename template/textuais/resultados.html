Introduo

Leitura de dados

O programa utilizado para analisar os dados disponibilizados em Excel ser o R Studio, verso 4.2.0, importados como um data frame (planilha), onde as colunas representam as variveis de estudo e cada linha representa um hospital dos Estados Unidos no perodo de 1975-1976.

Anlise Exploratria

Estatsticas Descritivas

Uma maneira fcil de obter algumas estatsticas descritivas das variveis em estudo atravs do comando summary(), que retorna as estatsticas mnimo, quartis, mdia e mximo. Para medir a variabilidade, utilize as funes var() e sd() para obter a varincia e o desvio padro outra forma de avaliar essas estatsticas atravs do grfico boxplot.
